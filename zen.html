Requirements

"Where possible, we would like to see mostly " CSS 1 & 2 " Usage. " CSS 3 & 4 " should be limited to widely-supported element only, or strong fallbacks should be provided. The CSS Zen Garden is about functional, practical " CSS " and not the latest bleeding-edge tricks viewable by 2% of the browsing public. The only real requirement is your " CSS " validates."

"Luckily, designing this way shows how well various browsers have implemented " CSS " by now. When sticking to the guidelines you should see farily consistent results across most modern browsers. Due to the sheer number of user agents on the web these days - especially when you factor in mobile - pixel - perfect layouts may not be possible across every platform. That's okay, but do test in as many as you can. Your design should work in at least IE9+ and the latest chrome, FireFox, iOS and Android browsers (run by over 90% of the population)."
